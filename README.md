## Install (One time only)

1. Install Node
2. Install Yeoman
```
sudo npm install -g yo
sudo npm install -g bower
sudo npm install -g grunt-cli
```
3. In this project root, run:
```
npm install
bower install
```
4. Proceed to Build and development

## Build & development
